Supporting code for:
G. Blaney*, F. Ivich*, et al., "Dual-ratio approach for detection of point 
fluorophores in biological tissue", JBO, In Rev. 2023
https://doi.org/10.48550/arXiv.2305.14436

- run_all.m
    This MATLAB script will call all subsequent scripts in the correct 
    order to recreate the figures presented in the paper.

Note: This code expects that MCXLAB
(http://mcx.space/wiki/index.cgi?Doc/MCXLAB) is installed and in MATLAB's
path

For additional information contact: Giles.Blaney[at]tufts.edu
